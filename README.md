# 🚧 Challenges & Learning Experience

One of the most rewarding aspects of building this project was the opportunity to learn and work with new technologies.

This was my first time using **Three.js**, **React Three Fiber**, and **React Three Drei** in a real-world project. Initially, understanding concepts such as 3D scenes, cameras, lighting, meshes, materials, and animations within the React ecosystem was quite challenging.

To overcome these challenges, I explored the official documentation, studied community examples, and experimented with different implementations until I gained a solid understanding of how these technologies work together.

Although learning these libraries required additional time and effort, successfully integrating them into the project significantly improved my understanding of modern web graphics, interactive animations, and frontend performance optimization.

This experience not only helped me complete the project but also strengthened my confidence in learning new technologies and applying them effectively in real-world applications.

# 📖 Project Overview

Raco AI is a modern web application that visually demonstrates how data flows through an AI-powered analytics pipeline.

Rather than presenting static information, the application delivers an immersive storytelling experience where users explore the complete lifecycle of data—from ingestion and AI analysis to actionable insights.

The project combines smooth animations, scroll-driven interactions, and interactive 3D graphics to create a premium user experience while maintaining a clean, modular, and scalable architecture.

# 🏗 Technical Approach

The project follows a modular component-based architecture using the Next.js App Router.

Each major section of the application is separated into reusable components, making the project easier to maintain and extend.

Animations are powered by Framer Motion, while all 3D graphics are rendered using Three.js through React Three Fiber. Styling is handled entirely with Tailwind CSS to maintain consistency and responsiveness across devices.

# 🎨 Animation & Interaction Decisions

## Hero Section

The hero section features a custom typewriter animation that continuously cycles through key platform capabilities. This creates an engaging first impression while drawing attention to the application's primary features.

Behind the content, a lightweight particle system built with React Three Fiber provides subtle depth without affecting readability or performance.

---

## Scroll-Locked Pipeline

Instead of allowing users to scroll through the content normally, the pipeline section introduces a guided storytelling experience.

Each mouse wheel interaction advances the user through one stage of the AI workflow, ensuring every step receives attention before moving forward.

Users can also jump directly to any stage by clicking the stage cards, providing both guided and manual navigation.

---

## Interactive 3D Section

The final section contains an animated wireframe object rendered with Three.js.

Continuous rotation, subtle cursor interaction, and floating information badges create a modern visual identity while keeping performance lightweight and responsive.
